QuickAdmin
===============

> 运行环境要求PHP7.1+，兼容PHP8.0。

基于 ThinkPHP 6 二次开发的一个基础后台管理模板

## 安装

~~~
composer create-project lifetime/quick-admin
~~~

## 对应前端模板

从仓库拉取代码
~~~
git clone https://github.com/LifeTimeNine/quick-admin-backstage.git -b v2
~~~

进入项目目录
~~~
cd quick-admin-backstage
~~~

安装依赖
~~~
npm install
~~~

运行
~~~
npm run dev
~~~
